## 高可用 React 服务端渲染 -- 郭达峰

### 问题（renderToString）

- node is single-threaded

- synchronous rendering

- CPU bound

### 解决方式

- NODE_ENV=production ：400ms -> 105ms

- Babel plugins（transform-react-inline-elements）：105ms -> 70ms

- Airbnb's Hypernova(优先尝试SSR，失败或超时降级到浏览器渲染)：Oversize Page
